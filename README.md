### Local Setup (Windows)
```
1. Clone the repo
2. cd MindChart
3. yarn
4. Star the monogdb server at default port 27017
5. cd packages/client
6. yarn start (To start the frontend server)
7. cd ../server
8. yarn dev (To start the backend server)
```
(Create an issue if these are not working for you)

### Tech Stack
